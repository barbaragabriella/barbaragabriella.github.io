What is this project about?

Smart and Safe Home (SSH) was created with the porpuse to control lighting, air conditioning inside home and secure the front door/gate against intruders as well as monitoring home in case of fire. The aim is to build a smart home with focus on safety and energy consumption.
If we control every light and air conditioner in a home, we will make sure no energy will be wasted just because someone forgot to turn off lights/air condioner when left home. Also, the system will make sure only allowed individuals will enter the house through face recognition. Whenever someone not allowed tries to be recognized, the system will alert the owner send a intruder message with a picture.
To make sure the user will be able to interact with the system, we also developed a mobile app which will allow the user to turn on the lights before getting home or turn them off when leaving and set up a temperature for the air conditioner. Also, the user will be able to monitor the presente, temperature, humidity and smoke data.
